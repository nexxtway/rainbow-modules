# Overview

`EmailPasswordSignUpForm` component is a fully functional Firebase 'email and password flow' sign up form.
It will notify the errors occurred during the sign up process through app level messages.

This component HAVE TO be used under a properly configured `RainbowFirebaseApp` component.

### Internationalization FormattedMessage keys

-   EmailPasswordSignUpForm.name: label of the name element
-   EmailPasswordSignUpForm.email: label of the email element
-   EmailPasswordSignUpForm.password: label of the password element
-   EmailPasswordSignUpForm.namePlaceholder: placeholder of the name element
-   EmailPasswordSignUpForm.emailPlaceholder: placeholder of the email element
-   EmailPasswordSignUpForm.passwordPlaceholder: placeholder of the password element
-   EmailPasswordSignUpForm.signup: text of the submit button
