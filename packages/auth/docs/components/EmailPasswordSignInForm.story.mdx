# Overview

`EmailPasswordSignInForm` component is a fully functional Firebase 'email and password flow' sign in form.
It will notify the errors occurred during the sign up process through app level messages.

This component MUST TO be used under a properly configured `RainbowFirebaseApp` component.

### Internationalization FormattedMessage keys

-   EmailPasswordSignInForm.email: label of the email element
-   EmailPasswordSignInForm.password: label of the password element
-   EmailPasswordSignInForm.emailPlaceholder: placeholder of the email element
-   EmailPasswordSignInForm.passwordPlaceholder: placeholder of the password element
-   EmailPasswordSignInForm.login: text of the submit button
