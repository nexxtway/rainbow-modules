# Overview

`useUpdateDoc` hook uses [react-query/useMutation](https://react-query.tanstack.com/guides/mutations)
hook behind the scene. The hook will return an object with a `mutate` async function that will update a document in a Firestore collection.

### Usage.

```js
const { mutate, isLoading } = useUpdateDoc();
const onUpdate = async (values) => {
    await mutate('/books/JLaSkUit8rIIqqXQvav1', values);
};

<Button label="Update Book" onClick={onUpdate} isLoading={isLoading} />;
```

This is end-to-end implementation of `how to` Add/Remove/Update on Firestore.
