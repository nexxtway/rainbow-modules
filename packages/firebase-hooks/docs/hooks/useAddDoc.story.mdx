# Overview

`useAddDoc` hook uses [react-query/useMutation](https://react-query.tanstack.com/guides/mutations)
hook behind the scene. The hook will return an object with a `mutate` async function that will add a document to a Firestore collection.

### Usage.

```js
const { mutate, isLoading } = useAddDoc('/books');
const onSubmit = async (values) => {
    await mutate(values);
};

<Button type="submit" label="Add Book" onClick={onSubmit} isLoading={isLoading} />;
```

This is end-to-end implementation of `how to` Add/Remove/Update on Firestore.
