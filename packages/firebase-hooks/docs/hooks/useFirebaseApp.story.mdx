# Overview

`useFirebaseApp` hook will return the app object passed to `RainbowFirebaseApp`.
This hook takes cares of retrieving the value from the context for you.

### This is a basic example.

This example shows how to get the `app` object and use it to sign-in as an anonymous user.

```js
const app = useFirebaseApp();
await app.auth().signInAnonymously();
```
