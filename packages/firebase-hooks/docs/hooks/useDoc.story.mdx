# Overview

`useDoc` hook will subscribe to a firestore doc by providing the path e.g `/books/id`.
It will return an tuple with [data, isLoading].

### This is a basic example.

In this example we are subcribing to a `/books/id` doc.

```js
const [data, isLoading] = useDoc({
    path: '/books/id',
});
```
