# Overview

`useRemoveDoc` hook uses [react-query/useMutation](https://react-query.tanstack.com/guides/mutations)
hook behind the scene. The hook will return an object with a `mutate` async function that will remove a document from a Firestore collection.

### Usage.

```js
const { mutate, isLoading } = useRemoveDoc();
const onRemove = async (id) => {
    await mutate(`/books/${id}`);
};

<Button label="Remove" onClick={() => onRemove('JLaSkUit8rIIqqXQvav1')} isLoading={isLoading} />;
```

This is end-to-end implementation of `how to` Add/Remove/Update on Firestore.
