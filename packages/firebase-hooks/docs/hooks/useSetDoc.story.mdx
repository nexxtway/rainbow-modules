# Overview

`useSetDoc` hook uses [react-query/useMutation](https://react-query.tanstack.com/guides/mutations)
hook behind the scene. The hook will return an object with a `mutate` async function that will update a document in a Firestore collection.

### Usage.

```js
const { mutate, isLoading } = useSetDoc();
const onSet = async (data) => {
    await mutate({ path: '/books/JLaSkUit8rIIqqXQvav1', data, options: { merge: true } });
};

<Button label="Set Book" onClick={onSet} isLoading={isLoading} />;
```

This is end-to-end implementation of `how to` Add/Remove/Update on Firestore.
