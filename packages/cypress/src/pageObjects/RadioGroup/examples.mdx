-   ##### How to check an option

```js
import { RadioGroup } from '@rainbow-modules/cypress/pageObjects';

const radioGroup = new RadioGroup('#radioGroup-selector');
radioGroup.getInput(0).check();
```

-   ##### How to check an option scrolling if it is not visible

```js
import { RadioGroup } from '@rainbow-modules/cypress/pageObjects';

const radioGroup = new RadioGroup('#radioGroup-selector');
radioGroup.getInput(0).check({ scrollBehavior: 'center' });
```

-   ##### How to test the error message

```js
import { RadioGroup } from '@rainbow-modules/cypress/pageObjects';

const radioGroup = new RadioGroup('#radioGroup-selector');
radioGroup.expect('error', 'This field is required');
```

-   ##### How to test the value

```js
import { RadioGroup } from '@rainbow-modules/cypress/pageObjects';

const radioGroup = new RadioGroup('#radioGroup-selector');
radioGroup.expect('value', 'radioThree');
```
