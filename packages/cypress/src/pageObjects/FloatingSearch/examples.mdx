-   ##### How to type a search query

```js
import { FloatingSearch } from '@rainbow-modules/cypress/pageObjects';

const floatingSearch = new FloatingSearch('#floating-search-selector');
floatingSearch.input.type('Hello world!');
```

-   ##### How to clear the search

```js
import { FloatingSearch } from '@rainbow-modules/cypress/pageObjects';

const floatingSearch = new FloatingSearch('#floating-search-selector');
floatingSearch.clear();
```

-   ##### How to close

```js
import { FloatingSearch } from '@rainbow-modules/cypress/pageObjects';

const floatingSearch = new FloatingSearch('#floating-search-selector');
floatingSearch.close();
```
