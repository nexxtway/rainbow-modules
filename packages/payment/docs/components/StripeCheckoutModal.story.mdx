# Overview

`StripeCheckoutModal` is a component that allows you to enter information from a credit or debit card, so you can execute a specific action. That action can be to save the credit/debit card information or pay directly with the card.

### Internationalization FormattedMessage keys

-   StripeCheckoutModal.setupIntent.title: modal title when variant is `setupIntent`
-   StripeCheckoutModal.paymentIntent.title: modal title when variant is `paymentIntent`
-   StripeCheckoutModal.errorMessage: error message
-   StripeCheckoutModal.cardholderLabel: label of the cardholder input
-   StripeCheckoutModal.cardholderPlaceholder: placeholder of the cardholder input
-   StripeCheckoutModal.cardInformationLabel: label of the card information input
-   StripeCheckoutModal.cancelButtonLabel: label of cancel button
-   StripeCheckoutModal.setupIntent.okButtonLabel: label of ok button when variant is `setupIntent`
-   StripeCheckoutModal.paymentIntent.okButtonLabel: label of ok button when variant is `paymentIntent`
