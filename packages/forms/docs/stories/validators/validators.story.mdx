# Overview

The `@rainbow-modules/forms` provides a set of ready-to-use validators which can be directly
provided to the `Field` component as in the next example:

```jsx
import { Field } from 'react-final-form';
import { isRequired } from '@rainbow-modules/forms';

<Field name="name" validate={isRequired()} />;
```

It is possible to combine more than one validator on the same field:

```jsx
import { Field } from 'react-final-form';
import { composeValidators, isRequired, maxLength } from '@rainbow-modules/forms';

<Field name="name" validate={composeValidators(isRequired(), maxLength(20))} />;
```
