# Icons

Rainbow icons are delightful, beautifully crafted symbols for common actions and items.

## Installation

Install the icons using npm.

```
$npm install @rainbow-modules/icons

```

Install the icons using yarn.

```
$ yarn add @rainbow-modules/icons

```
