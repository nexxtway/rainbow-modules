# Overview

`isHexRgbaOrHslaColor` checks if the given value is a valid color in hex, rgb(a) or hsl(a) format.

```js
isHexRgbaOrHslaColor('#FF00FF');
// returns true

isHexRgbaOrHslaColor('#FF0');
// returns true

isHexRgbaOrHslaColor('#FF0045FF');
// returns true

isHexRgbaOrHslaColor('#FF0F');
// returns true

isHexRgbaOrHslaColor('rgb(240, 10, 34)');
// returns true

isHexRgbaOrHslaColor('rgba(240, 10, 34, 0.1)');
// returns true

isHexRgbaOrHslaColor('rgba(240 10 34 / .1)');
// returns true

isHexRgbaOrHslaColor('hsl(23.0, 100%, 34%)');
// returns true

isHexRgbaOrHslaColor('hsl(23.0, 100%, 34%, .1)');
// returns true

isHexRgbaOrHslaColor('hsla(270 60% 50% / .15)');
// returns true

isHexRgbaOrHslaColor('#FF');
// returns false

isHexRgbaOrHslaColor('#FFFFF');
// returns false

isHexRgbaOrHslaColor('rgb(100, 100)');
// returns false

isHexRgbaOrHslaColor('rgba(100, 100, 100)');
// returns false

isHexRgbaOrHslaColor('hsl(100, 100)');
// returns false
```
