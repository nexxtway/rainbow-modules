# Overview

`AuditLogs` is a component to show logs stored in a Firestore collection. It comes with several filter like
severity and date filter. It offers the possibility to download the logs in CSV format.

The stored logs should comply with the following schema:

```json
{
    "severity": "debug" | "info" | "warning" | "error",
    "date": Timestamp,
    "textPayload": "Text description of the log",
    "labels": {
        "labelName": "Label value"
    }
}
```
