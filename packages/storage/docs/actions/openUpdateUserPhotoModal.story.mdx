# Overview

This hook is like the `useState`, but besides being reactive it store the data in the local storage. It receive a key string value which indicates where to set and get the data in the local storage.

## Usage

```js
const Foo = () => {
    const [state, setState] = useStorageState({ key: 'my-unique-key' });
    return (
        <div>
           <h1>{state}</h1/>
           <Button label="Switch Theme" onClick={state === 'light' ? 'dark' : 'light'} />
        </div>
    );
}
```

## API

| Property     | Description                                                                                         | Type   | Default   |
| ------------ | --------------------------------------------------------------------------------------------------- | ------ | --------- |
| key          | This represents a string (must be unique) to identify where the value will be in the local storage. | string | undefined |
| defaultValue | A default value to use when there is no value in the local storage under the key passed.            | any    | undefined |
