# Overview

`CopyToClipboardButton` A button with modern approach to copy text to clipboard. Would try to use execCommand with fallback to IE specific clipboardData interface.

### Internationalization FormattedMessage keys

-   CopyToClipboardButton.clickToCopy: text for the tooltip that describes the action of copy to clipboard
-   CopyToClipboardButton.copied: text for the tooltip that describes that the text was copied to the clipboard
