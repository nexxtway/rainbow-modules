### Installing from npm

---

Rainbow Modules is available as an [npm package](https://www.npmjs.com/package/rainbow-modules).

```bash
$ npm install rainbow-modules --save
```

or `yarn` package manager.

```bash
$ yarn add rainbow-modules
```
